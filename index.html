Bubble Sort

Bubble sort is a basic sorting algorithm that works by repeatedly going through a list of numbers and comparing adjacent pairs. If a pair is out of order (e.g., the first number is greater than the second), it swaps them. This process is repeated until the entire list is sorted. It's called "bubble sort" because smaller numbers "bubble" or move towards the beginning of the list with each iteration. Although bubble sort is easy to understand and implement, it's not very efficient for large lists since it takes a long time to complete.
